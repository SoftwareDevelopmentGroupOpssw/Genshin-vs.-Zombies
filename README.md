# Genshin vs. Zombies

这是一个类似于植物大战僵尸的游戏。其中使用了一部分植物大战僵尸的资源和原神游戏角色的图片，如有侵权请联系QQ：407021500.
在这个游戏中包含八位原神角色，同时添加了《原神》这款游戏中类似的现有的元素反应系统，具体反应效果可在游戏内帮助文件查看。

下载Release版后，双击PlantsVSZombies.exe即可开始游戏。
